Domain Monitor - Documentation

Stack

  • Node.js
  • Express (Backend)
  • Vanilla HTML/CSS/JS (Frontend)

Installation

install dependencies:

npm install express axios dotenv

Run the server:

node server.js

Flow Overview

  1. Server serves static files from /public.
  2. Frontend requests domain data from /api/domains.
  3. Backend loads domains from domains.js and calculates:
    • Expiry dates
    • Days remaining
    • Name server changes
  4. Frontend renders:
    • Table of domains
    • Dashboard cards (expiring soon, total domains, etc.)
  5. User can:
    • Add single or bulk domains
    • Delete domains
    • Refresh individual or all domains
    • Export CSV

Endpoints

  • GET /api/domains - Retrieve all domains
  • POST /api/domains - Add new domain
  • DELETE /api/domains/:domain - Delete a domain
  • POST /api/refresh - Refresh all domains
  • POST /api/refresh/:domain - Refresh single domain
  • /api/export/csv - Export domains as CSV

Frontend Features

  • Responsive dark-mode UI
  • Premium modern dashboard cards
  • Sortable table by expiry date
  • Inline and bulk domain additions
  • Delete and refresh buttons per row
  • CSV export

Notes

  • All dates are formatted as MMM D YYYY HH:mm:ss
  • Domains expiring soon are highlighted in dashboard cards
  • Bulk upload accepts comma-separated domain names
